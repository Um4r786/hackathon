# MatIntel Simulator

## Windows 

1) Set up a python venv by running **"python -m venv venv"** in the hackathon directory
2) Activate the venv by running **"venv/Scripts/activate"**
3) Install the required packages by running **"pip install -r requirements.txt"**
4) Run the program by running **"python main.py"**
5) The web-based application will run on **http://127.0.0.1:5000**
6) Navigate to the website to use the program.

# Mac

1) Set up a python venv by running **"python3 -m venv venv"** in the hackathon directory
2) Activate the venv by running **"source venv/bin/activate"**
3) Install the required packages by running **"pip3 install -r requirements.txt"**
4) Run the program by running **"python3 main.py"**
5) The web-based application will run on **http://127.0.0.1:5000/**
6. Navigate to the website to use the program.
